Cell2D is a Java library for making 2D games, designed to simplify and speed up development by handling common organizational tasks. It is based on and incorporates Slick2D.

Features

  • Convenient and precise event scheduling
  • Compartmentalization of behaviors into states for both objects and the game as a whole
  • Object-level control of the flow of time
  • Simple loading and management of graphics, music, and sound effects
  • Image manipulation

An extension called Cell2D Level for making 2D action game levels will be available shortly.
